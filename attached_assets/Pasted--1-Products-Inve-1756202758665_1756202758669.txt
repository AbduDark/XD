> الواجهة احترافية تدعم **دارك مود**. ### 🗂️ الموديولات الأساسية: 1. **إدارة المخزون (Products/Inventory)** * أصناف محددة: (سماعات أذن - سماعات - شاحن - ماوس - ميكات - ليدر - اوتو جي - جراب - وصلة مايكرو - وصلة تايب - إكسسوار - ستاند - سكرينة - ايربودز - كمبيوتر - باور بنك - أخرى). * إضافة/تعديل/حذف الأصناف. * تتبع الكميات + تنبيهات عند قرب انتهاء المخزون. 2. **نظام الفواتير (Invoices)** * إنشاء فاتورة مبيعات. * ربط الفاتورة بالمخزون. * طباعة الفاتورة (A4 أو حرارية POS). * البحث عن الفواتير السابقة. 3. **المرتجعات (Returns)** * إمكانية إرجاع منتج من فاتورة سابقة. * تعديل الكميات في المخزون آلياً. * تسجيل سبب المرتجع. 4. **قسم الصيانة (Repairs)** * صيانة **هاردوير** و **سوفت وير**. * بيانات العميل (اسم - تليفون). * الجهاز ونوع العطل. * تكلفة الإصلاح. * حالة الإصلاح (قيد العمل - تم الإصلاح - تم التسليم). 5. **قسم التحويلات المالية (Cash Transfers)** * تحويلات: فودافون كاش، اتصالات كاش، أورانج كاش، أكسس كاش، كروت فكة. * يتم إدخال المعاملات يدوياً (إضافة/تعديل/حذف). * تقارير مفصلة لهذه المعاملات. 6. **التقارير (Reports)** * تقارير مبيعات يومية/شهرية/سنوياً. * تقارير الأرباح والخسائر. * تقارير المخزون. * تقارير الصيانة. * تقارير التحويلات. * **رسوم بيانية (Charts.js أو ApexCharts)** في صفحة الداشبورد والتقارير. 7. **إدارة المستخدمين (Users Management)** * تسجيل دخول (Login). * أدوار متعددة: * **Super Admin**: التحكم الكامل بالمستخدمين (إضافة/تعديل/حذف/تحديد صلاحيات). * **Admin**: إدارة المخزون والفواتير والتقارير. * **موظف**: صلاحيات محدودة (مثلاً: فقط إضافة فواتير). * عند تسجيل الدخول: * إذا كان Super Admin → يحوله لصفحة التحكم بالمستخدمين. * إذا كان Admin أو موظف → يحوله للداشبورد. 8. **النسخ الاحتياطي (Backup System)** * نسخ احتياطي لقاعدة البيانات (يومياً + عند الطلب). * إمكانية تحميل النسخة كـ ملف SQL. --- ### 🛠️ الإعدادات الأولية: * إنشاء **Super Admin** افتراضي: * Email: alhussiny@super.com * Password: alhussiny55555 * إنشاء **Admin** افتراضي: * Email: alhussiny@admin.com * Password: admin@1234 --- ### 🎨 الواجهة (UI/UX): * لغة عربية كاملة (RTL). * تصميم احترافي بواجهة **Laravel Breeze أو Jetstream** + Tailwind. * **Dark / Light Mode Toggle**. * Dashboard مع رسوم بيانية توضح: * إجمالي المبيعات. * إجمالي المرتجعات. * إجمالي الأرباح. * المخزون المتبقي. * تحويلات الرصيد. --- ### 💡 ملاحظات إضافية: * حماية البيانات (Authentication + Authorization). * صلاحيات متدرجة للمستخدمين (Policies أو Spatie Laravel Permission). * كل صفحة لها فلترة (بحث حسب التاريخ -
